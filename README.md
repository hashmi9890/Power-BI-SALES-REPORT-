# Power-BI-SALES-REPORT- 
1. 📁 Data Source
Built using Excel/CSV files as the primary data source, simulating real-world sales data for analysis and visualization practice.

3. 📈 Key Visuals & Features
Includes interactive sales trend charts to track performance over time, a regional/map view for geographic sales distribution, and a product/category breakdown to analyze top-performing segments.

5. 👥 Target Audience
Designed for business analysts looking to explore sales insights, identify patterns, and support data-driven decision-making through visual reporting.

7. 🚧 Project Status
This is a demo/learning project built to practice Power BI skills including data modeling, DAX measures, and dashboard design. Feel free to explore, fork, or use it as a reference.
